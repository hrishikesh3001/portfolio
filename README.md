# 🖥️ OS Portfolio (Windows XP Inspired)

A fully interactive portfolio built as a simulated operating system, inspired by Windows XP.  
This project transforms a traditional portfolio into an immersive desktop experience.

---

## 🌐 Live Demo

👉 https://portfolio-six-gamma-88.vercel.app/

---

## 🚀 Features

- 🟢 Boot → Login → Desktop flow
- 🪟 Window system (open, close, minimize, focus)
- 📂 Start Menu with Programs (XP-style UI)
- 📝 Notepad with file creation & saving
- 🗑️ Recycle Bin system (delete & restore files)
- 🧮 Scientific Calculator (UI + keyboard support)
- 🎨 Paint application (canvas drawing)
- 🌤️ Weather + Date + Time widget
- 🔊 Sound effects for interactions
- 📄 Resume viewer (opens in browser)

---

## 🛠️ Tech Stack

- React
- JavaScript
- CSS
- React-RND (window management)

---

## 🎯 Purpose

This project demonstrates:

- Advanced UI/UX design
- System-like architecture in frontend applications
- State management across multiple interactive components
- Building non-traditional, immersive user experiences
