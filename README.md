# 🖥️ OS Portfolio (Windows XP Inspired)

A fully interactive portfolio built as a simulated operating system.

## 🚀 Features

- Boot → Login → Desktop flow
- Window system (open, close, minimize)
- Start Menu with Programs
- Notepad with file saving
- Recycle Bin system
- Scientific Calculator
- Paint application (canvas drawing)
- Weather + Date + Time widget
- Sound effects (click interactions)

## 🛠️ Tech Stack

- React
- JavaScript
- CSS
- React-RND (window system)

## 🎯 Purpose

This project demonstrates:

- UI/UX design skills
- System-like architecture in frontend
- Interactive application design

## 📦 Run Locally

```bash
npm install
npm run dev
```
